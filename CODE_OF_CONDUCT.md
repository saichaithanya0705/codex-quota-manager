# Code of Conduct

## Our Standard

This project expects respectful, professional collaboration.

Examples of behavior that contribute to a healthy community:

- Being direct without being hostile
- Respecting differing viewpoints and experience levels
- Giving actionable, technical feedback
- Taking responsibility for mistakes and fixing them quickly
- Focusing discussion on the project and the work

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks
- Discriminatory language or conduct
- Publishing private information without permission
- Bad-faith disruption of issues, pull requests, or discussions
- Any conduct that would reasonably make participation unsafe

## Scope

This applies within project spaces and in public interactions representing the project.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

For serious issues, contact the maintainer through the security policy in [SECURITY.md](./SECURITY.md).
